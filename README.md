# Transductive Few-shot Learning with Meta-Learned Confidence
Pytorch code for *Transductive Few-shot Learning with Meta-Learned Confidence.*[pdf](https://arxiv.org/abs/2002.12017)

## Requirements
* Python 3.6
* Pytorch 1.3.1
